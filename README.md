# 基于 Mobile OpenARM 的出门物品准备助手系统

## 项目简介

本项目设计一套面向家庭场景的出门物品准备助手系统，基于 Mobile OpenARM 实现室内自主导航、物品识别、机械臂抓取与智能提醒。

用户在出门前给出需要携带的物品清单，机器人根据清单在家庭环境中寻找目标物品，并将物品放置到玄关整理区。同时，系统能够结合天气、交通、日程和用户习惯进行辅助判断，例如在下雨时主动提醒并准备雨伞，在交通拥堵时提醒用户提前出门。

本项目不局限于校园场景，而是面向真实家庭生活，强调服务机器人走入社会后的实际应用价值。

## 核心功能

* **物品清单输入**：支持用户通过语音、手机端或控制界面输入出门物品清单。
* **多模态物品识别**：结合视觉识别、电子标签和用户习惯模型，提高物品查找准确率。
* **自主导航搜索**：机器人根据家庭地图移动到玄关、客厅、卧室、书房等区域进行搜索。
* **机械臂抓取放置**：识别目标物品后，使用机械臂完成抓取、搬运和放置。
* **智能决策提醒**：根据天气、交通、日程等信息主动推荐额外物品或出行建议。
* **任务状态反馈**：实时更新物品准备状态，并向用户反馈任务完成情况。

## 系统流程

```text
用户输入物品清单
        ↓
任务解析与清单结构化
        ↓
天气 / 交通 / 日程 / 用户习惯分析
        ↓
生成最终准备清单
        ↓
物品位置预测与搜索路径规划
        ↓
Mobile OpenARM 自主导航到目标区域
        ↓
视觉识别 + 电子标签确认目标物品
        ↓
机械臂抓取并放置到玄关整理区
        ↓
任务状态更新与用户提醒
```

## 系统架构

```text
用户交互层
├── 语音输入
├── 手机 / 网页端清单管理
└── 任务状态反馈

任务决策层
├── 清单解析模块
├── 天气 / 交通 / 日程分析模块
├── 用户习惯学习模块
└── 任务规划模块

├── RGB-D 视觉识别模块
├── 电子标签识别模块
├── 物品位置预测模块
└── 坐标变换模块

机器人执行层
├── 室内导航模块
├── 避障模块
├── 机械臂运动规划模块
├── 夹爪控制模块

```

## 关键技术

* Mobile OpenARM 移动操作平台
* ROS 机器人操作系统
* 室内建图与自主导航
* RGB-D 视觉识别与三维定位
* RFID / BLE / NFC 等电子标签辅助识别
* 机械臂运动规划与夹爪控制
* 用户习惯学习与规则决策
* 天气、交通、日程信息接入

## 典型应用场景

用户准备出门时输入：

```text
帮我准备钥匙、钱包和耳机。
```

系统检测到当天有雨后，主动询问：

```text
检测到今天可能下雨，建议携带雨伞，是否加入清单？
```

用户确认后，机器人依次寻找钥匙、钱包、耳机和雨伞，并将物品放置到玄关整理区。任务完成后，系统反馈：

```text
钥匙、钱包、耳机和雨伞已准备完成。当前路况较拥堵，建议提前 15 分钟出门。
```

## 项目特色

本项目的特色在于将机器人从单一抓取任务扩展为完整的家庭出门辅助流程。系统不仅能够执行用户给出的物品清单，还能结合外部环境和用户习惯进行主动提醒，体现服务机器人在真实生活场景中的智能化和实用性。
